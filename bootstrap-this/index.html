Benefits of Using Bootstrap

Mobile First

Bootstrap supports and follows the standards of a mobile-first approach using a flexbox grid and a twelve column layout to align content for all screen sizes.

A Time Saver!

With Bootstrap, you can save up a lot of time when working on a website. It's a perfect framework for any web developer out there that enjoys using frameworks.

It has Components!

Bootstrap comes with so many different types of components. Many components include buttons, cards, accordions, pagination, list groups, and carousels.

It has Utilities!

Bootstrap also includes many utilities. This includes borders, backgrounds, shadows, interactions, displays, overflows, text, sizing, position and spacing.

How to Use Bootstrap?

Step 1: Go to getbootstrap.com
Step 2: Click get the Get Started button
Step 3: Scroll down until you see the Starter Template
Step 4: Click the copy button or highlight all the code and right click to copy.
Step 5: Paste the code in your code editor.
Step 6: Save the code and open it in the Live Server to see Hello world!
